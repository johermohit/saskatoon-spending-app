# Saskatoon Spending Heatmap

## Description
This application provides a 3D spatial visualization of infrastructure spending by the City of Saskatoon. It maps procurement data to geographic coordinates and extrudes spending amounts to create an interactive data story.

## Deployment
To deploy this application to Vercel for free:
1. Initialize a Git repository in the `saskatoon-spending-app` folder.
2. Push the contents to a new GitHub repository.
3. Connect your GitHub repository to [Vercel](https://vercel.com).
4. Vercel will automatically detect the `public/index.html` as the entry point. Ensure the build settings point to the `public` directory if prompted.
